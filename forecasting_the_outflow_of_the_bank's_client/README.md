# forecasting_the_outflow_of_the_bank's_client
Прогнозирование оттока клиентов Банка.

На основе данных из банка определить клиента, который может уйти.

| Название проекта | Описание | Используемые библиотеки | 
| :---------------------- | :---------------------- | :---------------------- |
| [Прогнозирование оттока клиентов Банка](forecasting_the_outflow_of_the_bank's_client.ipynb) | Из банка стали уходить клиенты. Каждый месяц. Немного, но заметно. Банковские маркетологи посчитали: сохранять текущих клиентов дешевле, чем привлекать новых.Нужно спрогнозировать, уйдёт клиент из банка в ближайшее время или нет. Предоставлены исторические данные о поведении клиентов и расторжении договоров с банком. | *pandas, matplotlib, scikit-learn* |
