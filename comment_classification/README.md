# comment_classification
Обучение модели классификации комментариев.

Определение токсичности комментариев пользователей.

| Название проекта | Описание | Используемые библиотеки | 
| :---------------------- | :---------------------- | :---------------------- |
| [Обучение модели классификации комментариев](comment_classification.ipynb) | Интернет-магазин запускает новый сервис. Теперь пользователи могут редактировать и дополнять описания товаров, как в вики-сообществах. То есть клиенты предлагают свои правки и комментируют изменения других. Требуется инструмент, который будет искать токсичные комментарии и отправлять их на модерацию. | *pandas, nltk, tf-idf* |
